What is Full Stack Development?

Full Stack means building both the parts of a website. The front-end that users see and the back-end that runs on the server. A full stack developer should know how to work with HTML, CSS, and JavaScript for the front, and also server code like PHP, Node.js or Python for the back-end.

HTML

HTML is the standard markup language used to create and structure content on the web. HTML is used to structure a web page. Below is an example of a basic HTML page: 

<!DOCTYPE html>
<html>
<head>
  <title>Test Page</title>
</head>
<body>
  <h1>Hello!</h1>
  <p>This is a test page.</p>
</body>
</html>

CSS

CSS, which stands for Cascading Style Sheets, is a style sheet language used to define the presentation and visual appearance of a web page.. For example: 

body {
  background-color: lightblue;
  font-family: Arial;
}

h1 {
  color: darkblue;
}
